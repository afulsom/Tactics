SD4 A4: Roguelike Gold

Known Issues:
	Enemies can get stuck on each other.

How to Use:

	Run by opening Roguelike.exe.

	Keyboard Controls:
		Pressing the arrow keys moves the player in that direction.
		Pressing 'Space' makes the player rest for a turn.
		Pressing 'F1' opens the developer console.
		Pressing 'C' opens the stats screen.

	Pressing 'Escape' opens the menu and quits the game on the main menu.
